# Visual Style Guide - Legal Writing 101 Simulation

## Brand Identity

### JUS Brand Colors
**Primary Palette:**
- JUS Blue: #1E3A8A (RGB: 30, 58, 138)
- Professional Gray: #6B7280 (RGB: 107, 114, 128)
- Clean White: #FFFFFF (RGB: 255, 255, 255)

**Secondary Palette:**
- Success Green: #10B981 (RGB: 16, 185, 129)
- Warning Amber: #F59E0B (RGB: 245, 158, 11)
- Error Red: #EF4444 (RGB: 239, 68, 68)
- Accent Blue: #3B82F6 (RGB: 59, 130, 246)

**Usage Guidelines:**
- JUS Blue: Headers, primary buttons, branding elements
- Professional Gray: Body text, secondary elements
- Success Green: Positive feedback, completion states
- Warning Amber: Caution states, attention items
- Error Red: Error states, critical alerts

### Typography

**Primary Font: Montserrat**
- Headers (H1-H3): Montserrat Bold
- Subheaders (H4-H6): Montserrat SemiBold
- Usage: Titles, navigation, important UI elements

**Secondary Font: Open Sans**
- Body text: Open Sans Regular
- Emphasis: Open Sans SemiBold
- Usage: Paragraphs, descriptions, content

**UI Font: Inter**
- Buttons: Inter Medium
- Labels: Inter Regular
- Usage: Interface elements, forms, controls

**Font Sizes:**
- H1: 32px (Scene titles)
- H2: 24px (Section headers)
- H3: 20px (Subsection headers)
- Body: 16px (Standard text)
- Small: 14px (Captions, labels)
- Tiny: 12px (Fine print, metadata)

## Layout Standards

### Grid System
**Desktop (1920x1080):**
- 12-column grid
- 80px margins
- 20px gutters
- Safe area: 1760x920px

**Tablet (1024x768):**
- 8-column grid
- 40px margins
- 20px gutters
- Safe area: 944x688px

**Mobile (375x667):**
- 4-column grid
- 20px margins
- 16px gutters
- Safe area: 335x627px

### Component Spacing
- Large spacing: 40px (between major sections)
- Medium spacing: 24px (between related elements)
- Small spacing: 16px (between form elements)
- Micro spacing: 8px (between closely related items)

- Minimal saturation boost
- Professional retouching (subtle)

## UI Element Standards

### Buttons
**Primary Button:**
- Background: JUS Blue (#1E3A8A)
- Text: White (#FFFFFF)
- Font: Inter Medium, 16px
- Padding: 12px 24px
- Border radius: 6px
- Hover: Darker blue (#1E40AF)

**Secondary Button:**
- Background: Transparent
- Border: 2px solid JUS Blue
- Text: JUS Blue
- Font: Inter Medium, 16px
- Padding: 10px 22px
- Border radius: 6px
- Hover: Light blue background

**Disabled Button:**
- Background: Professional Gray (#6B7280)
- Text: White with 60% opacity
- No hover effects
- Cursor: not-allowed

### Form Elements
**Input Fields:**
- Border: 1px solid Professional Gray
- Background: White
- Padding: 12px 16px
- Border radius: 4px
- Focus: JUS Blue border, blue shadow

**Checkboxes:**
- Size: 20px x 20px
- Border: 2px solid Professional Gray
- Check color: JUS Blue
- Border radius: 3px

**Radio Buttons:**
- Size: 20px diameter
- Border: 2px solid Professional Gray
- Selected: JUS Blue fill
- Border radius: 50%

### Progress Indicators
**Progress Bar:**
- Height: 8px
- Background: Light gray (#F3F4F6)
- Fill: JUS Blue (#1E3A8A)
- Border radius: 4px

**Scene Navigation:**
- Breadcrumb style
- Current scene: JUS Blue
- Completed: Success Green

## Interactive Element Standards

## Accessibility Standards

### Color Contrast
- Text on white: Minimum 4.5:1 ratio
- Large text: Minimum 3:1 ratio
- UI elements: Minimum 3:1 ratio
- Focus indicators: High contrast outline

### Interactive Elements
- Minimum touch target: 44px x 44px
- Clear focus indicators
- Keyboard navigation support
- Screen reader compatibility

### Visual Hierarchy
- Clear heading structure (H1-H6)
- Consistent spacing patterns
- Logical reading order
- Sufficient white space

## Animation Standards

### Transitions
**Duration:**
- Micro interactions: 150ms
- Standard transitions: 300ms
- Complex animations: 500ms
- Scene transitions: 800ms

**Easing:**
- Standard: ease-out
- Bouncy effects: cubic-bezier(0.68, -0.55, 0.265, 1.55)
- Smooth: cubic-bezier(0.4, 0, 0.2, 1)

### Loading States
- Skeleton screens for content loading
- Progress indicators for long operations
- Smooth fade-ins for new content
- Consistent loading animations

## Quality Assurance

### Image Requirements
- Resolution: 2x for retina displays
- Format: PNG for transparency, JPG for photos
- Optimization: Compressed for web delivery
- Alt text: Descriptive for accessibility
