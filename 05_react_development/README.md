# Legal Writing 101 Simulation - React Application

## Quick Start

1. **Install Dependencies**
   ```bash
   cd 05_react_development
   npm install
   ```

2. **Start Development Server**
   ```bash
   npm start
   ```
   Opens http://localhost:3000

3. **Build for Production**
   ```bash
   npm run build
   ```
